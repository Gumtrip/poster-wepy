# poster-wepy
小程序-海报
------
主要功能 <br>
1.第一次登陆，根据手机号码绑定微信openid,以后自动登陆 <br>
2.定时刷新登陆token <br>
3.海报列表，海报列表筛选,海报详情 <br>
4.会员中心，修改个人信息 <br>
